# 🕹️ JOSH_ARTISAN_DEV_V1

Welcome to **JOSH_ARTISAN_DEV_V1**, an interactive, retro-arcade developer portfolio designed to showcase full-stack engineering skills through an immersive 8-bit terminal experience.

Instead of a standard static page, this portfolio acts as a simulated arcade system, complete with a typing boot-up sequence, CRT monitor flicker, CSS parallax backgrounds, and custom retro UI components.

## 🚀 Features

* **Terminal Boot Sequence:** A Vue.js powered typewriter effect that simulates a system initialization before revealing the main interface.
* **CRT & Phosphor Glow:** Custom pure CSS animations that recreate the scanlines, slight flicker, and text-glow of a classic 1980s arcade monitor.
* **Bootstrap Utility Hijacking:** A fully customized Bootstrap 5 implementation that forces all standard utility classes to obey a strict "Arcade Red" and "8-Bit Gold" color palette.
* **Fully Responsive:** Media queries specifically engineered to ensure large arcade typography and code blocks scale perfectly down to mobile devices without breaking the layout.
* **Dynamic Data:** Project showcases, skills arrays, and dynamic modals rendered cleanly via Vue.js reactive state.

## 🛠️ Tech Stack

* **Frontend:** Vue.js 3, Bootstrap 5, Pure CSS3 Animations
* **Backend Framework:** Laravel (PHP)
* **Build Tool:** Vite
* **Typography:** 'Press Start 2P' (Google Fonts)
